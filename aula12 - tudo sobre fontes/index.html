Tudo referente a fontes no CSS

Uso de fontes por @import

Quando queremos importar fontes que não estão por padrão no computador, devemos nos atentar a forma que queremos importar, podemos tanto baixar e importar do computador local como também, podemos importar usando url.

Exemplo: Usando fonte sem baixar

O título do blog, usou-se a fonte Orbitron com o seguinte comando: @import url('https://fonts.googleapis.com/css2?family= Orbitron&display=swap');

Exemplo: Baixando e usando fonte

Os subtítulos do blog, usou-se a fonte:Quicksand com o seguinte passo a passo:

  1. Baixar a fonte no Google Fonts (você pode baixar diretamente por aqui a que foi usada no site: Clicando aqui.

  2. Depois disso, instale o arquivo na pasta raiz de seu arquivo html (recomendo criar uma pasta chamada fonts, e inserir todas as fontes nela.)

  3. Após inserido, usa-se o seguinte comando: @font-face { font-family: 'Quicksand'; src: url('fonts/Quicksand-Light.ttf') format('truetype'); }

  4. Insira em seu arquivo CSS.

  5. Pronto, fonte instalada e pronta para usar :)
